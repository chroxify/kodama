# Kodama Monorepo

Turbo workspace for Kodama avatar generation.

## Package

- `kodama-id`
  - `kodama-id` (core deterministic APIs)
  - `kodama-id/react` (React components built on top of core)

## App

- `web`: Next.js playground that consumes `kodama-id/react`

## Commands

```bash
bun install
bun dev
bun run check-types
bun run lint
```
